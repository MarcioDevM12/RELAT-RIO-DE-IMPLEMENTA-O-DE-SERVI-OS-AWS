# RELAT-RIO-DE-IMPLEMENTA-O-DE-SERVI-OS-AWS
Estudo de caso completo de otimização de custos AWS. Este relatório apresenta o processo de implementação de ferramentas na empresa Abstergo Industries , realizado por Márcio Dias Dos Santos. O objetivo do projeto foi elencar 3 serviços AWS, com a finalidade de reduzir custos imediatamente.
